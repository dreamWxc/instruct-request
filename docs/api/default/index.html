# 基本配置

# rest

数据是否展开,默认axios的数据返回将是 一个 responseData 里面包含请求成功的 status data headers 等配置信息，默认我们最需要的便是 data 后台返回的内容 查看详情

# 参数: Boolean

示例展示

# 不展开的数据格式:

{
    status:200,
    headers:{},
    data:{
        code:200,
        msg:'成功',
        data:[]
    },
    ...option
}

# 展开的数据格式:

{
    code:200,
    msg:'成功',
    data:[]
}

# message

弹窗配置用于请求的弹窗提示等工作，多用于插件 查看详情

# 参数: Object


# responseCode

成功的状态码,配置后台返回的状态,配合codeKey使用 查看详情

# 参数: Array<number>


# codeKey

校验后台返回的状态字段,配合responseCode使用 查看详情

# 参数: String


# customCheck

以上不符合需求，自定义设置校验 查看详情

# 参数: Function
